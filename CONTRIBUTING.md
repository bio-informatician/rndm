# Contributing

Thanks for considering a contribution!

## Setup

```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -e ".[dev]"
```

## Checks

- Run tests: `pytest`
- Lint: `ruff check .`

## Pull requests

- Keep changes focused and well-described
- Add/adjust tests for behavioral changes
